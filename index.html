📖 Getting Google Credentials
Follow these steps to connect your company's Google Drive:
  1. Go to console.cloud.google.com
  2. Create/select a project → Enable Google Drive API & Google Picker API
  3. APIs & Services → Credentials → Create API Key
  4. Create OAuth 2.0 Client ID → Web application
  5. Add your domain to Authorized JavaScript Origins
  6. Find your invoices folder in Drive → copy the ID from the URL
Folder ID is the part after /folders/ in the Google Drive URL.
Example: drive.google.com/drive/folders/1BxiMVs0XRA5...
